## 質問 17/50

db1 という名前のAzure SQL Databaseがあります。
db1 内のすべてのデータが保存時に暗号化されていることを確認する必要があります。
何を使用する必要がありますか?

### 選択肢

- 常に暗号化されています
- Azure SQL監査
- Microsoft Defender for SQL
- [正解] Transparent Data Encryption (TDE)

### 解説

TDE は、保存データを暗号化するAzure SQL Database機能です。 ディスクに保存されたすべてのデータは暗号化されます。
監査は保存時のデータ暗号化を実行するために使用されるのではなく、診断情報の収集のために使用されます。
sql のDefenderは、データベースの潜在的な脆弱性を軽減し、異常なアクティビティを検出するために使用されます。
Always Encrypted は、使用中または転送中のデータを保護します。
