## 質問 5/50

Azure SQL Managed Instanceにデータベースがある。
インスタンスで現在実行中のすべてのセッションの待機情報を分析し、待機数が最も多いクエリを選択する必要があります。  
クエリを実行する必要がある動的管理ビューはどれですか?

### 選択肢

- [選択・不正解] sys.dm_db_wait_stats
- sys.dm_exec_query_profiles
- sys.dm_exec_query_statistics_xml
- [正解] sys.dm_exec_sessions_wait_stats

### 解説

sys.dm_exec_sessions_wait_stats ビューは、セッションごとに実行されたスレッドによって検出されたすべての待機に関する情報を返します。 このビューを使用して、Microsoft SQL Server セッションおよび特定のクエリとバッチに関するパフォーマンスの問題を診断できます。
sys.dm_exec_query_profiles ビューは、クエリの実行中にリアルタイムのクエリの進行状況を監視し、sys.dm_exec_query_statistics_xml ビューは実行中の要求のクエリ実行プランを返し、sys.dm_db_wait_stats ビューは操作中に実行されたスレッドによって検出されたすべての待機に関する情報を返します。
