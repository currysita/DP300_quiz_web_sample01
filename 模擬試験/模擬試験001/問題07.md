## 質問 7/50

Azure SQL データベースがある。
ブロックとデッドロックのパフォーマンスの問題をトラブルシューティングする必要があります。
どの機能を使用する必要がありますか?

### 選択肢

- Azure SQL Database の台帳
- [正解] 拡張イベント
- Microsoft Defender for SQL
- クエリストア

### 解説

拡張イベントは、パフォーマンスの問題の調査とトラブルシューティングに使用できるブロックおよびデッドロックのメトリックをキャプチャできます。
クエリ ストアは、ブロックおよびデッドロックのメトリックをキャプチャせず、データベースの脆弱性をAzure SQL Defenderに表示し、Azure SQL Database台帳はデータベースの状態を増分的にキャプチャします。
