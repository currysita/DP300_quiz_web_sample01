# DP300_quiz_web_sample01
DP-300の問題集をもとにクイズを作成しました。ブラウザのキャッシュにデータを保存しますのでブラウザさえあれば実行可能です

## キーボードショートカット（問題画面）

| キー | 動作 |
|---|---|
| `→` / `L` / `N` | 次の問題へ |
| `←` / `H` / `P` | 前の問題へ |
| `A` 〜 `F` | 選択肢を選択 / トグル |
| `Enter` / `Space` | 採点する（複数選択問題） |
