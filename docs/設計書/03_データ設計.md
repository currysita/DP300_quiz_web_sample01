# データ設計

## 問題集 JSON スキーマ

ファイル例: `data/q001-050.json`

```json
{
  "id": "q001-050",
  "title": "DP-300 問題集 第1回",
  "range": "問題 1〜50",
  "questions": [
    {
      "id": 2,
      "question": "Azure 仮想マシンで実行されている SQL Server データベースのサイズを確認するために、どのツールを使用すべきですか？",
      "type": "single",
      "choices": [
        { "label": "A", "text": "Azure Monitor" },
        { "label": "B", "text": "SQL Server Management Studio（SSMS）" },
        { "label": "C", "text": "Azure SQL Analytics" },
        { "label": "D", "text": "SQL Assessment" }
      ],
      "answer": "A",
      "explanation": "Azure Monitor は Azure VM 上の SQL Server に対して、データベースのサイズ・ストレージ使用量・パフォーマンスメトリクスの監視が可能。",
      "explanation_table": [
        { "label": "A", "text": "DB サイズ・ストレージ・パフォーマンスメトリクスを監視可能", "correct": true },
        { "label": "B", "text": "SQL Server 管理ツール。Azure 環境統合監視には不向き", "correct": false },
        { "label": "C", "text": "PaaS の SQL 監視（DB/MI）向け。IaaS には最適でない", "correct": false },
        { "label": "D", "text": "構成評価とベストプラクティス確認が目的。サイズ監視ではない", "correct": false }
      ],
      "deprecated": false,
      "deprecation_note": ""
    }
  ]
}
```

---

## フィールド定義

### 問題集ルート

| フィールド | 型 | 説明 |
|---|---|---|
| `id` | string | ファイル名と一致するID（URLパラメータで使用） |
| `title` | string | 問題集名（選択画面に表示） |
| `range` | string | 出題範囲の説明（選択画面に表示） |
| `questions` | array | 問題の配列 |

### 問題オブジェクト

| フィールド | 型 | 説明 |
|---|---|---|
| `id` | number | 問題番号（元の問題集の番号） |
| `question` | string | 問題文 |
| `type` | string | `"single"`（1択）固定。将来の拡張用 |
| `choices` | array | 選択肢の配列（2〜4個） |
| `answer` | string | 正解の選択肢ラベル（例: `"A"`） |
| `explanation` | string | 解説テキスト |
| `explanation_table` | array | 各選択肢の解説行（選択肢テーブル） |
| `deprecated` | boolean | 廃止・出題注意フラグ。true の場合、問題文上部に警告バッジを表示する |
| `deprecation_note` | string | 廃止理由・代替情報（`deprecated: true` の場合） |
| `skip` | boolean | 出題スキップフラグ。true の場合、問題一覧から除外して出題しない |
| `skip_note` | string | スキップ理由（`skip: true` の場合に記載） |

### 選択肢オブジェクト（choices）

| フィールド | 型 | 説明 |
|---|---|---|
| `label` | string | 選択肢記号（`"A"` 〜 `"D"`） |
| `text` | string | 選択肢テキスト |

### 解説テーブル行（explanation_table）

| フィールド | 型 | 説明 |
|---|---|---|
| `label` | string | 選択肢記号 |
| `text` | string | 解説文 |
| `correct` | boolean | 正解かどうか |

## 問題集インデックスファイル（data/index.json）

問題集の一覧は `app.js` に埋め込まず、`data/index.json` で管理する。  
アプリ起動時に `fetch()` で読み込む。

```json
[
  {
    "id": "q001-050",
    "title": "DP-300 問題集 第1回",
    "range": "問題 1〜50",
    "file": "data/q001-050.json"
  },
  {
    "id": "q051-100",
    "title": "DP-300 問題集 第2回",
    "range": "問題 51〜100",
    "file": "data/q051-100.json"
  }
]
```

### フィールド定義

| フィールド | 型 | 説明 |
|---|---|---|
| `id` | string | 問題集の識別子。URLパラメータで使用 |
| `title` | string | 問題集名（選択画面に表示） |
| `range` | string | 出題範囲の説明（省略可） |
| `file` | string | 問題集JSONのパス（`data/` からの相対パス） |

### 問題集追加手順

1. 問題集JSONファイルを `data/` に配置する
2. `data/index.json` にエントリを追加する

`app.js` の編集は不要。
