# 企画書：資格取得補助プラットフォーム

**作成日**: 2026-05-02  
**ステータス**: 草案

---

## 1. 背景・課題

現在、DP-300（Azure Database Administrator）の問題集を静的Webアプリとして開発・運用している。
開発を通じて以下の課題・拡張ニーズが見えてきた。

- 問題集のみ対応で、用語集・参考書・教育資料など他の学習コンテンツを格納する仕組みがない
- 1つの資格（DP-300）しかサポートしていない
- コンテンツが増えた際に、現在のディレクトリ構成では管理しきれない
- 学習の進捗や履歴を記録・継続する手段がない

---

## 2. 目的・ゴール

複数の資格試験に対応し、問題集だけでなく多様な学習コンテンツを一元管理・提供できる**資格取得補助プラットフォーム**を構築する。

### ゴールイメージ

- 学習者が1つのサイトで「問題集・用語集・参考資料」をまとめて利用できる
- 資格の種類を増やしても、同じ仕組みで対応できる
- コンテンツ追加・更新の作業負荷が低い

---

## 3. 対象ユーザー

- IT系資格の取得を目指す学習者（主に個人）
- 複数の資格を並行して学習している人
- 体系的な参考資料を手元に置きながら問題演習をしたい人

---

## 4. 機能概要（案）

### 4.1 コンテンツ種別

| コンテンツ種別 | 概要 |
|---|---|
| 問題集 | 選択式の演習問題。現行機能の継承・拡張 |
| 用語集 | 試験頻出キーワードの定義・解説集 |
| 教育用資料 | 概念図・構成図・チートシートなど |
| 参考書（要約） | 試験範囲のトピックをまとめた読み物コンテンツ |

### 4.2 資格サポート

- 資格ごとにコンテンツをグルーピングして管理する
- 初期対応: DP-300（既存コンテンツを移行）
- 拡張候補: AZ-900、SC-900、AZ-104 など Azure 関連資格

### 4.3 ナビゲーション

- トップページで資格を選択する
- 資格ごとのポータルページから各コンテンツ種別にアクセスする

### 4.4 将来的な機能（スコープ外 / 検討事項）

- 学習進捗の記録（ローカルストレージまたはサーバーサイド）
- 問題の正答率・苦手分野の可視化
- ユーザーアカウント管理

---

## 5. 想定技術スタック（案）

現行プロジェクトの制約（GitHub Pages・ビルドツールなし）を継承するかどうかも含めて検討が必要。

| 項目 | 現行 | 次期（案） |
|---|---|---|
| ホスティング | GitHub Pages（静的） | GitHub Pages 継続 または Vercel/Netlify |
| フロントエンド | HTML / JS / Bootstrap | 同左 または Vue/React 導入も検討 |
| データ管理 | JSON ファイル | JSON 継続 または Headless CMS |
| バックエンド | なし | なし（静的継続）または 軽量API |

**当面の方針案**: 静的サイト構成を継続し、ディレクトリ構成とデータ構造の設計を見直すことで拡張性を確保する。

---

## 6. スコープ外（今回は対象としないもの）

- ユーザー登録・ログイン機能
- 有料コンテンツ・課金機能
- スマートフォンアプリ（iOS / Android ネイティブ）
- AIによる問題自動生成
